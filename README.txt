Para desplegar la página:

- Descargar folder completo
- Abrir index.html en algún browser

Notas:

El ícono mostrado del tipo de tarjeta de crédito (AMEX, VISA, MC) depende del primer dígito del número de la tarjeta; si el número empieza con '3' es AMEX, si empieza con '4' es VISA y si empieza con '5' es MC.



To display the page:

- Download complete folder
- Open up index.html in some browser

Notes:

The shown credit card icon (AMEX, VISA, MC) depends on the first digit of the credit card number; if the number starts with a '3' it's AMEX, if it starts with '4' it's VISA, and if it starts with '5' it's MC.
